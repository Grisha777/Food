# Рецепты блюд

### В данном репозитории представлен Pet-проект "Еда"

Технологии: HTML, CSS, JavaScript, Vue, публичное API (https://www.themealdb.com/api.php)

Реализовано:
1) Оображения данных с публичного API в виде таблицы
2) Страница блюд, страница категорий, страница для каждого персонального блюда, страница добавления нового рецепта
3) Добавление нового рецепта, редактирование существующих
